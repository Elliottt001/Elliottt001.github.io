## 无语义化标签

| 标签      | 类型   | 用途                               | 语义性       |
|-----------|--------|----------------------------------|-------------|
| `<span>`  | 内联   | 包裹文本或内联内容，应用样式/脚本      | 无          |
| `<div>`   | 块级   | 布局分组，包裹块级内容               | 无          |
| `<i>`     | 内联   | 表示技术术语或外文短语（HTML5 语义化） | 弱语义      |
| `<b>`     | 内联   | 表示需注意的内容（HTML5 语义化）      | 弱语义      |
