## 基本骨架
```html
 <!DOCTYPE html>  <!--声明文档类型-->
<html>

    <head>  <!--head双标签-->
        <title>CosHub</title>  <!--title标签中一般不展示给用户-->
        <meta charset="utf-8">  <!--meta单标签，中间写内容，charset指定编码-->
    </head>

    <body>

    </body>

</html>
```

## 标题
```html
<h1> 到 <h6> 双标签 例如 <h1> </h1> <!--正斜杠,放在body中-->
```

快捷键：
```plaintext
h$*6 : 自动生成1~6级标签，其中后面的数字为n，代表自动生成1~n级标签
```
