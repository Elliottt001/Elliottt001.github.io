2. 软硬件接口
3. 性能
    - 算法
    - 编程语言
    - 处理器 / 内存
    - I/O

## 计算机思想精华

1. 抽象：用抽象简化设计
2. 时间：使常用情况加速，例如循环
3. 并行 parallelism：多核（并行CPU）
4. 流水线 pipelining
5. 预测 prediction：内存取数纳秒硬盘取数微秒，预取
6. 层次：按照层次理解系统。硬盘——内存——缓存cashe——寄存器register（速度由慢到快，容量大到小）
7. 可靠性
